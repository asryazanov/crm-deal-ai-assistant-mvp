# AI-Ассистент сделки для CRM

Локальный end-to-end MVP web-страницы для демонстрации AI-Ассистента в карточке сделки.

Ассистент:

- рассчитывает вероятность закрытия сделки по прозрачной rule-based методологии;
- определяет здоровье сделки: зеленая, желтая или красная зона;
- показывает ключевые риски;
- формирует рекомендации отдельно для менеджера и руководителя;
- работает локально без внешних AI API.

## Запуск

```bash
python3 -m http.server 8787
```

После запуска открыть:

```text
http://127.0.0.1:8787/
```

## Структура

- `index.html` — однофайловая web-страница: встроенные CSS, JS, scoring engine и демо-датасет.
- `data/deals.json` — тестовый датасет сделок для MVP.
- `docs/mini-tz.md` — мини-ТЗ и формула скоринга.
- `Axoft_Deal_Dataset_v1.0.xlsx` — исходный тестовый датасет.
- `Методология_оценки_сделки_Версия_2.1.docx` — исходная методология.
- `Предложение_по_запуску_пилотных_проектов_на_базе_ИИ_в_Аксофт_ИИ Помощник_в1.docx` — исходное описание пилота.

## MVP-ограничения

- Скоринг rule-based, без ML-калибровки.
- Рекомендации шаблонные, формируются по найденным рискам.
- Нет записи задач обратно в CRM.
- Нет авторизации и интеграции с MS Dynamics / 1С / Power BI.
